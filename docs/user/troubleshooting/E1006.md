# E1006 Details

## Message emitted:

`E1006`: There was no password defined, preemptively failed on device `{hostname}`.

## Description:

There was no password defined, preemptively failed on device `{hostname}`.

## Troubleshooting:

Ensure that the password is defined and accessible for the device `{hostname}`.

## Recommendation:

Likely the credentials class is not setup correctly, check configuration here: https://docs.nautobot.com/projects/plugin-nornir/en/latest/user/app_feature_credentials/.
