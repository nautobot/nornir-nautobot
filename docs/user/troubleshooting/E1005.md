# E1005 Details

## Message emitted:

`E1005`: There was no username defined, preemptively failed on device `{hostname}`.

## Description:

There was no username defined, preemptively failed on device `{hostname}`.

## Troubleshooting:

Ensure that the username is defined and accessible for the device `{hostname}`.

## Recommendation:

Likely the credentials class is not setup correctly, check configuration here: https://docs.nautobot.com/projects/plugin-nornir/en/latest/user/app_feature_credentials/.
